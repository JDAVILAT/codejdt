Frequently Asked Questions (FAQ)

WHAT DO I DO IF THE CODE IS INVALID?

If your code is not recognized by our system, verify that you have entered it correctly. If the problem persists, please contact us for help.

WHAT HAPPENS IF I LOST MY AUTHENTICATION CODE?

The authentication code is affixed to the product packaging. If you have lost it, we will not be able to verify the authenticity. We suggest that you always save the product packaging until you are sure of its authenticity.
